# aws-s3-portfolio-cloudfront

🌐 AWS S3 Portfolio Website with CloudFront
📌 Project Overview

This project demonstrates how I deployed my portfolio website using AWS cloud services. The website is hosted on Amazon S3 and delivered securely and efficiently using CloudFront.

This is part of my journey toward becoming an AWS Community Builder, where I am learning and sharing Cloud, DevOps, and AI concepts. 🚀

🛠️ Services Used
Amazon S3 – for static website hosting
Amazon CloudFront – for fast and secure content delivery
⚙️ Architecture

User → CloudFront (CDN + HTTPS) → S3 Bucket (Static Website)

🚀 Steps I Followed
Created an S3 bucket
Uploaded portfolio website files (HTML, CSS, JS)
Enabled static website hosting
Configured bucket permissions for public access
Created a CloudFront distribution
Connected CloudFront to S3
Enabled HTTPS for secure access
💡 Key Learnings
How to host static websites on AWS
Importance of CDN for performance and security
Basics of cloud-based deployment
Real-world architecture understanding
